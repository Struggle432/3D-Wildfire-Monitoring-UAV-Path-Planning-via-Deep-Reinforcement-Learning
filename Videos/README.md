# 视频文件夹
## 视频内容说明
1. **`ppo_monitoring.mp4` (PPO监测)**
   * 顶部实时显示当前 `Step`、`Coverage` 和 `Reward` 指标。
   * 同时展示 2D 平面投影与 3D 仿真环境。
2. **`comparison.mp4` (算法对比)**
   * 横向对比 4 种算法的灭火表现。
   * 采用 4×2 布局（第一行展示 2D 视角，第二行展示 3D 视角）。
3. **`multi_scenario.mp4` (多场景对抗)**
   * 左右分屏对比无人机群在不同环境下的鲁棒性。
   * 左侧：简单场景（单火源、无风）；右侧：复杂场景（三火源、强风）。
